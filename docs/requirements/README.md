# requirements

이 폴더는 요구사항 정리 문서를 저장한다.

예시:
- 기능 요구사항 초안
- 화면 목록
- 정책 메모
- 범위 정의
- 사용자 시나리오

권장 파일명 예시:
- feature-user-management.md
- admin-board-requirements.md

- perf-budget.json (성능 게이트 — [`docs/performance/`](../../docs/performance/README.md) 템플릿 복사본)
